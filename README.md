# Coursera Paragraph Cleaner

A simple web-based tool to remove AI assistant paragraphs from your Coursera assessments, ensuring your submissions are clean and authentic.

## Features

- **AI Detection**: Automatically identifies and removes AI assistant paragraphs
- **Smart Cleaning**: Removes unwanted content while preserving your original text
- **Easy Copy**: One-click copy to clipboard for cleaned text
- **Real-time Stats**: Track input/output characters and removed paragraphs
- **Secure & Private**: All processing happens locally in your browser

## How to Use

1. Open `index.html` in your web browser
2. Paste your Coursera assessment text into the input area
3. Click "Clean Text" to remove AI paragraphs
4. Review the cleaned output
5. Click "Copy Cleaned Text" to copy the result to your clipboard

## Browser Compatibility

Works in all modern web browsers with JavaScript enabled.

## Privacy

This tool processes all text locally in your browser. No data is sent to any server or stored anywhere.
